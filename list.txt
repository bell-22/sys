Makefile 구성: 
target: dependencies
	commands
dependencies(의존성) 파일들을 사용해 commands(명령어)를 실행해서 target(이름) 파일을 만듦
반복되는 파일/옵션을 변수로 정의해 관리!
자동변수: @ # 현재 타겟 이름, ^ # 현재 타겟의 모든 의존성 파일, < # 첫 번째 의존성 파일의 이름
Makefile 예시:
1. 변수 설정
CC=gcc, CFLAGS=-W -Wall, TARGET=hello, OBIS=main.o hello.o
2-1. 빌드 규칙: TARGET이라는 파일을을 모든(@) 의존성 파일(^=OBJS)을 사용해 생성
$(TARGET): $OBJS
	$(CC)$(CFLAGS) -o $@$^ 
2-2. 각 .o 파일 생성 규칙: main.c(소스코드)와 hello.c(헤더파일#include)로 main.o라는 중간파일을 생성
main.o:main.c hello.h
	$(CC)$(CFLAGS) -c $<
3. 정리 규칙:
.PHONY:clean
clean:
	rm -f $(OBJS)$(TARGET)
